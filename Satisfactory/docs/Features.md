# Features Overview

## Canvas And Navigation

- Grid-based planning canvas with X/Y coordinates.
- Drag and drop placement for rail features.
- Middle-mouse pan and Space+left-drag pan.
- Zoom control plus Fit View and Fit To Selection.

## Panels

- Build Palette on the left.
- Inspector on the right.
- Workspace panel for view toggles, zoom, history, and fit controls.
- Train Stations panel for quick navigation to station entities.
- Dockable Workspace and Train Stations panels on any side of the canvas.
- Drag-resizable dock panels with visible grip markers.

## Station Layouts

- Horizontal layouts with metadata on either side.
- Vertical layouts with metadata above or below the freight area.
- Per-station layout direction stored on the station itself.
- Connection points stay attached to the station border for the active layout.

## Editing Tools

- Inspector editing for selected stations, sections, signals, and junctions.
- View toggles for section labels, signal endpoints, directional indicators, and validation icons.
- Persistent panel docking and size preferences.
