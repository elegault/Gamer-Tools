# Quick Start

## Prerequisites

- Node.js 20 or newer.
- npm.

## Install

From the `Satisfactory/app` folder:

```bash
npm install
```

## Run Locally

Start the Vite dev server:

```bash
npm run dev
```

Open the local URL shown in the terminal, usually `http://localhost:5173`.

## Build For Production

```bash
npm run build
```

## Preview A Production Build

```bash
npm run preview
```

## Typical Workflow

1. Load or create a map.
2. Add stations, sections, signals, and junctions from the toolbar.
3. Select an object to edit its properties in the Inspector.
4. Dock the Workspace and Train Stations panels where they are easiest to use.
5. Use Fit View or Fit To Selection to reframe the canvas.
